# 🚀 Nexus – Business Collaboration Platform

A modern business collaboration platform built with **React, TypeScript, Vite, and Tailwind CSS**. This project was developed as part of an internship to enhance the existing Nexus platform by implementing collaboration, communication, scheduling, document management, security, and payment features.

## 🌐 Live Demo

**Vercel:** https://vercel.com/manahil-abid1/nexus

## 📂 GitHub Repository

https://github.com/Manahil-Abid-dev/Nexus

---

# 📌 Project Overview

Nexus is a platform that connects **Entrepreneurs** and **Investors** by providing tools to collaborate, communicate, schedule meetings, manage documents, and track business deals in a secure environment.

The project includes multiple dashboards, real-time styled interfaces, and responsive design for desktop and mobile devices.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Manahil-Abid-dev/Nexus.git
```

Move into project

```bash
cd Nexus
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
